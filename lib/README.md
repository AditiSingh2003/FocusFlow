# FocusFlow Architecture (lib/)

## Folder Tree

```
lib/
├─ core/
│  ├─ constants/
│  ├─ themes/
│  └─ utilities/
├─ data/
│  ├─ models/
│  ├─ repositories/
│  └─ local_db/
├─ logic/
│  ├─ bloc/
│  └─ services/
└─ presentation/
   ├─ screens/
   ├─ widgets/
   └─ animations/
```

## Guidelines

- **core**: constants, themes, cross-cutting utilities.
- **data**: models, repositories, local DB implementations.
- **logic**: BLoC/Cubit and domain services.
- **presentation**: screens, widgets, animations.
